\section{贝叶斯估计}
\subsection{贝叶斯公式的密度函数形式}
%@see: 《概率论与数理统计》（茆诗松、周纪芗、张日权）
设\(X,Y\)是两个连续型随机变量，
则\begin{equation*}
	p_{Y \vert X}(y \vert x)
	= \frac{
		p(x,y)
	}{
		p_X(x)
	}
	= \frac{
		p_{X \vert Y}(x \vert y) p_Y(y)
	}{
		\int p_{X \vert Y}(x \vert y) p_Y(y) \dd{y}
	},
\end{equation*}
其中\(p\)是\(X,Y\)的联合密度函数，
\(p_X,p_Y\)分别是\(X,Y\)的边缘密度函数，
\(p_{X \vert Y}\)是\(X\)关于\(Y\)的条件密度函数.

我们把\(p_Y\)称为\(Y\)的\DefineConcept{先验分布}，
把\(p_{Y \vert X}\)称为\(Y\)的\DefineConcept{后验分布}.

\(p_{X \vert Y}\)综合了总体信息和样本信息，
\(p_{Y \vert X}\)进一步综合了总体信息、样本信息和先验信息.
